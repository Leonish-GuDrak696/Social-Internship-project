# AI-Powered Cyber Threat and Fraud Detection System

## Overview

This project uses Machine Learning to detect phishing and fraudulent emails. It analyzes email content and classifies messages as safe or potentially malicious.

## Features

* Email phishing detection
* TF-IDF text vectorization
* Logistic Regression classifier
* Streamlit web interface
* Real-time message analysis

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Joblib
* Streamlit

## Dataset

The model was trained using the CEAS_08 phishing email dataset.

## Model Performance

* Algorithm: Logistic Regression
* Feature Extraction: TF-IDF
* Accuracy: 99.45%
